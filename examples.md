# Examples of Names Acknowledgment System (NAS)

This document provides examples of how to represent different name structures and preferences using the NAS notation.

## Iberoamerican Name

* Paulina Andrea Valenzuela Lagos **(n.n.s.s)**
* Paulina Andrea Valenzuela Lagos **(N.n.S.S)**

### Same person, different contexts (v.2)

The same name can carry different NAS renderings depending on the setting. There is no single "correct" one — each is a personal choice.

* *Legal / scientific:* Paulina A. Valenzuela Lagos **(N.n.S.S)** — full formality; *Andrea* shown as the initial "A.", both surnames emphasized.
* *Daily / casual email, Lagos written:* Paulina Valenzuela Lagos **(N.(n.)S.s)** — *Andrea* implied but not written; *Lagos* present but not required for address.
* *Daily / casual email, Lagos omitted (her actual signature):* Paulina Valenzuela **(N.(n.)S.(s.))** — *Lagos* isn't written at all here, so it takes the parenthetical `(s.)` instead of the bare `s` above.

Note: since 2022, Chilean law allows the maternal surname to be listed before the paternal one. NAS notation simply reflects whichever order the person currently presents — it records the *current* structure, not the reasoning or lineage behind it.

## Icelandic Name (patronymic / matronymic)

Icelanders use a given name plus a patronymic (father's name + *-son* / *-dóttir*), are addressed by their given name, and are alphabetized by it. The trailing unit is a patronymic (`p`), not an inherited family name.

* Björk Guðmundsdóttir **(N.p)** — "daughter of Guðmundur"; addressed as "Björk"
* Guðni Th. Jóhannesson **(N.n.p)** — middle name *Thorlacius* shown as the initial "Th."
* Jón Einarsson **(N.p)**

## Russian Name (patronymic middle)

Formal address is given name + patronymic. Casual address is the given name alone.

* Vladimir Vladimirovich Putin **(N.P.s)** (formal — "Vladimir Vladimirovich")
* Vladimir Vladimirovich Putin **(N.p.s)** (casual — "Vladimir")
